# Wilson

Wilson is a store website for 6 brands sell sport .

## Installation

Clone the git repo:

git clone https://github.com/dada-da/wilson
or download the ZIP file:

https://github.com/dada-da/wilson/blob/master/team3-wilson-eproject.zip

## Usage

Start the server:

npm start
And access the URL shown in the browser. By default it's:

http://localhost:5000/
but you can start the server listening on a differet port:

PORT=4444 npm start
